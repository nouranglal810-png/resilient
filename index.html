ℹ️ About GLOB AI

AI-Driven Cybersecurity Assistant

GLOB AI

Your AI-Powered Cybersecurity Guardian

GLOB AI is an AI-powered cybersecurity assistant designed to help users identify unsafe or malicious links and understand cybersecurity risks through a conversational interface. The project combines AI-based question answering with real-time URL safety analysis, presented in a secure, user-friendly chat environment.

🤖

AI Chatbot

Answers cybersecurity questions with educational explanations to improve cyber awareness.

🔗

Link Detection

Automatically detects and analyzes URLs for phishing, malware, and security threats.

📁

File Support

Image, video, and file attachment support with secure preview functionality.

🎤

Voice Input

Speech-to-text support for hands-free interaction and improved accessibility.

🎨

Modern UI

Glassmorphism-based design with animated backgrounds and responsive layouts.

Secure Design
