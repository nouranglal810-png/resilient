# 🔐 GLOB AI – AI-Driven Cybersecurity Assistant

## 📌 Overview
GLOB AI is an AI-powered cybersecurity assistant designed to help users identify unsafe or malicious links and understand cybersecurity risks through a conversational interface. The project combines AI-based question answering with real-time URL safety analysis, presented in a secure, user-friendly chat environment.

## 🧠 Key Features

### 🤖 AI Chatbot
- Answers general and cybersecurity-related questions
- Covers 20+ topics: Phishing, Malware, Ransomware, VPN, Encryption, Hacking, Dark Web, XSS, SQL Injection & more
- Smart synonym matching for natural language understanding
- Educational explanations to improve cyber awareness

### 🔗 Link Safety Detection
- Automatically detects URLs from user input
- Analyzes links using pattern-based threat intelligence
- Classifies links as: **Safe** ✅ | **Suspicious** ⚠️ | **Unsafe** 🚨
- Provides risk scores (0-100) and detailed findings

### 📁 Media & File Support
- Image, video, and file attachment support
- Secure preview without execution
- Prevents unsafe behavior from uploaded content

### 🎤 Voice Input
- Speech-to-text support via Web Speech API
- Allows hands-free interaction
- Improves accessibility

### 🎨 Secure & Modern UI
- Glassmorphism-based design with animated Particles.js background
- Cyber grid overlay effect
- Responsive for desktop and mobile devices
- Orbitron & Poppins fonts

## 🛠️ Tech Stack

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Glassmorphism UI with animations & transitions

### Libraries & Tools
- **Particles.js** – Animated background
- **Google Fonts** – Orbitron, Poppins
- **Web Speech API** – Voice input

## 🚀 How to Run
1. Clone the repository
2. Open `index.html` in a browser, or
3. Run with a local server:
```bash
npx serve . -l 3000
```
4. Open http://localhost:3000
